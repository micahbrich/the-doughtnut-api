Doughnuts API

Introduction

This is an example RESTful API around the most delicious thing in the entire world – doughnuts. You can do anything you'd expect to do with a RESTful API:

  • GET
  • POST
  • PUT
  • PATCH
  • DELETE

And furthermore, everything is set up to use both CORS and JSONP, so you should be able to access this information from any client.

Usage

This API is for example purposes, so feel free to modify, delete, and add information as necessary. Because everything's only stored in memory, you'll be able to PUT, DELETE, or POST and see the result as if the database were modified, but when you refresh or hit another endpoint it will be reset. Which means you really can't hurt it!

Keep it clean, keep it classy, keep it doughnuts.
